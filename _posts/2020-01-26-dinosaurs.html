I have been teaching myself Java through an app called SoloLearn. Yesterday I was given a prompt to create a program that takes the amount of a loan from a friend and outputs the total amount left after 3 months and 10% interest a month.

I took the instructions a little too literally and created an output showing what 10% of each payment and the amount left after each month.

I was proud of myself! I didn't think it would format correctly. I'm still learning how to use the print statements.
