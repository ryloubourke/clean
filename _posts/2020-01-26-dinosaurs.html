I was proud of myself! I didn't think it would format correctly. I'm still learning how to use the print statements.

This was really fun for me not only because the mental model of programming clicked a little more, but because I took it a step further. To save my work for later, I found a website explaining how to write the code in Notepad and compiling through the command line in Windows. I then ran the program in the command line.
